# Weather Temperature Prediction

## Project Overview

This project predicts air temperature using weather parameters and time-based features.

## Dataset

- Minute-level weather observations

## Features

- Air Pressure
- Humidity
- Wind Speed
- Rain
- Time Features

## Models Used

- Linear Regression
- Decision Tree
- Random Forest
- Gradient Boosting

## Best Model

Gradient Boosting Regressor

R² Score: 0.80

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## dataset link
https://www.kaggle.com/datasets/niharikanamade/weather-data-minute

## How to Run

pip install -r requirements.txt

streamlit run app.py
